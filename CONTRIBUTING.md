# Contribution Guidelines

Thank you for your interest in contributing to Discooptbot! Please follow these guidelines to ensure a smooth contribution process.

## How to Contribute
1. **Fork the repository**
2. **Create a new branch** for your feature or bug fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes and commit them**:
   ```bash
   git commit -m "Add a new feature"
   ```
4. **Push to your fork**:
   ```bash
   git push origin feature/your-feature-name
   ```
5. **Open a pull request** to the main repository for review.

## Code Standards
Please ensure your code adheres to our coding standards before submitting a pull request.

- Write clear, maintainable code
- Include comments and documentation where necessary
- Follow our project's style guide (link to style guide if available)

## Reporting Issues
If you find a bug or would like to propose an enhancement, please open an issue in the GitHub repository.

Thank you for contributing!
